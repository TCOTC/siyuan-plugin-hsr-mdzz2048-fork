# 思源笔记 文档高亮搜索 插件

> 注意：如果更新后无法打开搜索框，请重启一次思源笔记

#### 如何使用

启用后会在顶栏右边添加一个按钮，点击即可打开搜索。

另外还添加了快捷键 `Ctrl+Shift+Alt+F`，可以在快捷键设置自行修改。

只搜索当前页签内的文本，不包含界面和文档标题。
